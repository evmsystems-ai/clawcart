How It Works

Three steps from request to checkout

User Asks Agent
"Order school supplies
for 3rd grade under $50"
→
2
Agent Builds Cart
Searches products, compares
prices, assembles cart
→
3

share-a-cart.com/c/abc123

4

// Agent code - it's this simple
const cart = new ClawCart({ retailer: 'amazon' });
await cart.addItem({
query: "Crayons 24-pack",
quantity: 2
});
const shareUrl = await cart.share();
